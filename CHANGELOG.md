# 更新日志

## [0.1.0] - 2024-03-16

### 新增
- 初始版本发布
- 核心游戏循环：整备 → 部署 → 搜刮 → 撤离
- 完整的战斗系统
  - 身体部位伤害（头部/胸部/腹部/四肢）
  - 护甲减伤机制
  - 瞄准系统
- 装备系统
  - 武器：手枪、冲锋枪、突击步枪、狙击步枪、霰弹枪
  - 护甲：1-6级防弹衣
  - 头盔：2-4级头盔
  - 消耗品：绷带、急救包、手术包、止痛药、能量饮料
- 物品系统
  - 稀有度分级（灰色到金色）
  - 价值计算
  - 背包格子存储
- 地图系统
  - 工业区地图
  - 区域连接和移动
  - 撤离点机制
- 敌人系统
  - 多种敌人类型
  - AI巡逻行为
  - 战利品掉落
- 玩家成长
  - 等级系统
  - 经验值获取
  - 资金积累
