# 暗区文字 (Dark Zone: Text)

一款基于《三角洲行动-烽火地带》核心玩法设计的文字撤离射击游戏。

## 游戏简介

《暗区文字》将硬核撤离射击体验转化为文字冒险形式。玩家需要进入危险区域搜集高价值物资，然后成功撤离。死亡将丢失背包中所有物品，高风险高回报！

## 核心玩法

### 游戏循环
```
整备(基地) → 部署(进入地图) → 搜刮与战斗 → 撤离 → 交易与升级
```

### 核心机制

1. **撤离射击**：进入地图搜集物资，成功撤离才能保留战利品
2. **死亡惩罚**：死亡会丢失背包中所有物品（保险箱物品保留）
3. **风险决策**：多搜刮可能获得更多物资，但撤离点可能关闭
4. **装备系统**：武器、护甲、消耗品等完整装备系统

## 游戏系统

### 身体部位系统
- 头部（2.5x伤害倍率，致命）
- 胸部（1.0x）
- 腹部（0.9x）
- 四肢（0.7x，受伤会影响行动）

### 护甲减伤
```
最终伤害 = 基础伤害 × 部位倍率 × (1 - 护甲减伤率)
护甲减伤率 = max(0, (护甲等级 - 子弹穿透等级) × 0.2 + 0.1)
```

### 物品稀有度
| 稀有度 | 颜色 | 价值倍率 |
|--------|------|----------|
| 普通 | 灰色 | 1.0x |
| 优秀 | 绿色 | 1.5x |
| 稀有 | 蓝色 | 2.0x |
| 史诗 | 紫色 | 3.0x |
| 传说 | 金色 | 5.0x |

### 敌人类型
- 游荡者新兵（弱）
- 游荡者（普通）
- 老练游荡者（强）
- 佣兵士兵（精英）
- 佣兵精英（极强）
- 杀手Killa（BOSS）

## 操作指南

### 基地
- `1` - 开始行动
- `2` - 整理装备
- `3` - 查看仓库
- `4` - 休息恢复

### 行动中
- `m [编号]` - 移动到指定区域
- `s` - 搜索当前区域
- `e` - 尝试撤离
- `i` - 查看背包
- `p` - 查看状态
- `q` - 紧急撤离（丢失背包物品）

### 战斗中
- `a [部位]` - 攻击（head/body/legs）
- `r` - 换弹
- `h` - 使用医疗物品
- `f` - 撤退

## 安装运行

### 方式一：Docker部署（推荐）

```bash
# 克隆项目
git clone https://github.com/yourusername/text-game3.git
cd text-game3

# 构建并运行
docker compose up -d

# 访问游戏
# 浏览器打开 http://localhost:5001
```

### 方式二：直接运行

```bash
# 安装依赖
pip install flask

# 运行Web版本
python3 app.py

# 或运行命令行版本
python3 game.py
```

## 技术架构

- **语言**: Python 3

- **部署**: Docker支持

  - `app.py` - Flask Web服务
