# 暗区文字 (Dark Zone: Text)

一款基于《三角洲行动-烽火地带》核心玩法设计的文字撤离射击游戏。

## 游戏简介

《暗区文字》将硬核撤离射击体验转化为文字冒险形式。玩家需要进入危险区域搜集高价值物资，然后成功撤离。死亡将丢失背包中所有物品，高风险高回报！

## 核心玩法

### 游戏循环
```
整备(基地) → 部署(进入地图) → 搜刮与战斗 → 撤离 → 交易与升级
```

### 核心机制

1. **撤离射击**：进入地图搜集物资，成功撤离才能保留战利品
2. **死亡惩罚**：死亡会丢失背包中所有物品（保险箱物品保留）
3. **风险决策**：多搜刮可能获得更多物资，但风险也更大
4. **装备系统**：武器、护甲、头盔、背包、消耗品等完整装备系统

## 游戏系统

### 身体部位系统（简化三部位）
- 头部（2.0x伤害倍率，致命）
- 胸部（1.0x，标准伤害）
- 腿部（0.7x，受伤影响移动）

### 状态异常
- **流血**：受伤30%概率触发流血，每回合持续掉血，需使用绷带或医疗包止血
- **部位破损**：部位HP降为0时破损，需使用手术包修复

### 护甲系统
- 护甲覆盖头部和胸部，腿部无护甲保护
- 护甲耐久度会随受伤降低，可使用修理包修复
- 6级护甲体系（PACA软甲到Slick重型防弹衣）
- 头盔体系（SSH-68到FAST头盔）

### 物品稀有度
| 稀有度 | 颜色 | 价值倍率 |
|--------|------|----------|
| 普通 | 灰色 | 1.0x |
| 优秀 | 绿色 | 1.5x |
| 稀有 | 蓝色 | 2.0x |
| 史诗 | 紫色 | 3.0x |
| 传说 | 金色 | 5.0x |

### 武器系统
- P226手枪、MP5冲锋枪、AK-74N突击步枪
- M4A1卡宾枪、莫辛纳甘步枪、M870霰弹枪

### 敌人类型
- 游荡者新兵（弱）
- 游荡者（普通）
- 老练游荡者（强）
- 佣兵士兵（精英）
- 佣兵精英（极强）
- 杀手Killa（BOSS）

### 撤离方式
- **开放撤离点**：直接撤离
- **等待撤离点**：需等待一段时间
- **付费撤离点**：支付费用撤离（直升机坪 $5000）
- **丢弃背包撤离**：丢弃所有背包物品撤离

### 商店系统
- 武器商店：购买各类武器
- 护甲商店：购买防弹衣
- 头盔商店：购买头盔
- 背包商店：升级背包容量（12格到48格）
- 消耗品商店：医疗物品和护甲修理工具

### 存档系统
- 支持保存/读取游戏进度
- 基于JSON的存档格式

## 安装运行

### 方式一：Docker部署（推荐）

```bash
# 克隆项目
git clone https://github.com/yourusername/text-game3.git
cd text-game3

# 构建并运行
docker compose up -d

# 访问游戏
# 浏览器打开 http://localhost:5001
```

### 方式二：直接运行

```bash
# 安装依赖
pip install flask

# 运行Web版本
python3 app.py

# 或运行命令行版本
python3 game.py
```

## 技术架构

- **语言**: Python 3
- **Web框架**: Flask
- **架构**: 模块化设计
- **部署**: Docker支持
- **存档**: JSON文件存储
- **核心模块**:
  - `game.py` - 游戏核心逻辑（数据定义、游戏类、存档系统）
  - `app.py` - Flask Web服务（API、Session管理）
  - `templates/index.html` - 前端界面（地图可视化、战斗UI）
